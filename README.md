## ULTRAVIOLET-ENGINE

Motor de conversión imagen → ASCII. Transforma fotografías en patrones de texto mediante análisis de luminancia y mapeo de caracteres.

### Stack
- React + Vite
- Canvas API para procesamiento de píxeles
- Algoritmo de conversión: brightness-to-character mapping

### Features
- Drag & drop interface
- Color/monochrome modes
- 20+ character sets (hiragana, blocks, braille, etc.)
- Dynamic sizing
- Export PNG/TXT
- Zoom functionality

### Cómo usar
1. Arrastra una imagen al área de drop
2. Selecciona charset y tamaño
3. Toggle color/mono
4. Export o zoom según necesites

### Execution
```bash
npm install
npm run dev
```
