# ULTRAVIOLET

A web-based image to ASCII art converter by medusahra.

Converts images into ASCII art with support for monochrome and color output.

## Features

- Drag & Drop interface
- Color and monochrome modes
- Multiple character sets
- Adjustable size output
- Save as PNG
- Zoom functionality

## Tech Stack

- React + Vite
- HTML Canvas API
